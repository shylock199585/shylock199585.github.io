nc

用来快速构建网络链接,常用来调试客户端程序。

参数

参数 描述
-i 设置数据包传送的时间间隔
-l 以服务器方式运行，默认为客户端运行
-k 重复接受并处理某个端口的所有链接
-p 以客户端运行时强制使用指定端口
-C 将CR和LF两个字符作为结束符
-u 使用UDP协议，默认为TCP协议
-X nc客户端余代理服务器通信默认为socks5协议
-z 扫描目标机器某个范围内服务是否开启
